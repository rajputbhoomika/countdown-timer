Countdown Timer Project

Features:
- Set countdown with title, date, and time
- Real-time countdown display
- Sound alert when countdown ends
- Theme switcher (light/dark)
- Animated countdown digits

How to Use:
1. Open index.html in a browser
2. Enter title, date, and time
3. Click Start to begin countdown
4. Toggle theme with button
5. Sound plays when countdown ends

Files:
- index.html
- style.css
- script.js
- alarm.mp3
